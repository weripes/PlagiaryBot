Инструкция к программе.

Для запуска бота потребуется:
Этап 1: api_id и api_hash
1.Получить api id.txt, открыть папку config, найти файл api_id, стереть всё что есть в файле и вставить ваш api id
2.Получить api hash, открыть папку config, найти файл api_hash.txt, стереть всё что есть в файле и вставить ваш api hash

Как получить api id и api hash?
Заходим на сайт https://my.telegram.org/, проходим авторизацию, заходим в API development tool. Нас попросят указать для чего нам нужен API, указываем например автоответчик, в поле ссылка пишем None. После завершения перед вами появятся api_id и api_hash.

Этап 2: Заполнение файлов
1.Открыть папку config, найти файл account_name.txt, стереть всё что есть в файле и вставить имя вашего аккаунта телеграм.
2.Открыть папку config, найти файл from_channel.txt, стереть всё что есть в файле и вставить название канала без @ с которого в будем собирать информацию.
3.Открыть папку config, найти файл to_channel.txt, стереть всё что есть в файле и вставить название канала без @ в который будем постить.

Этап 3: Запуск
1.Открываем cmd, вводим cd и путь к файлу
2.Запускаем файл main.py написав python main.py (Нужно установить python на ваш компьютер)
2.Вводим номер телефона, с этого аккаунта мы будем постить и собирать посты. Мы должны быть подписаны на канал с которого берем информацию, и быть администратором канала куда будем постить.
3.Вводим смс код
4.(Оптимально) Вводим пароль телеграм

Готово! Бот запущен! При публикации поста он уведомит вас в консоли и укажет время поста!

Для выключения бота, нажмите в консоли ctrl + c
